# case01（仮名加工情報）: 列ごとの情報特性・機微度分類

> **ステータス: 未着手（プレースホルダ）**

## 概要

直接識別性・間接識別性・機微度・情報分類・結合キー性を評価する。機微度分類は本教材の加工設計上の補助指標であることを明示する。

## 前提

- 対象: オンライン通販等の顧客データ・購買データを用いた仮名加工情報の事例
- 一次情報を確認した上で作成すること
- レポート記載事項と教材用の独自設定を明確に区別すること
