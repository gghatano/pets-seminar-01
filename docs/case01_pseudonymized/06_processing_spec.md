## 3つのルール

- **年代への一般化**: 基準日で年齢を算出し、「10代」〜「80代」に区分（80歳以上は「80代」に丸め）。
- **住所の一般化**: 「都道府県 + 市区町村」まで残し、丁目・番地以降を削除。政令市は「◯◯市◯◯区」まで（例: 横浜市青葉区）。
- **整理番号**: 会員IDと1対1で対応する整理番号（例: `R000001`）を付与し、**顧客・購買・アクセスの3テーブルで同じ整理番号**を使って個人単位の履歴結合を維持。元の会員IDは削除する。
