## 2. 年代への一般化ルール〔報告準拠〕

- 年齢 = 基準日の年 − 生年（教材では `REFERENCE_DATE = 2026-07-01` を使用）。
- 「10代」〜「80代以上」に区分（レポート図表1-3）。本教材では表示を「NN代」に統一し、80歳以上は「80代」に丸める〔簡略〕。

## 3. 住所の一般化ルール〔報告準拠〕

- 「都道府県 + 市区町村」までを残し、丁目・番地以降を削除。
- 政令市は「◯◯市◯◯区」まで（例: 横浜市青葉区、さいたま市浦和区）を1単位とする〔簡略〕。

## 4. 整理番号〔報告準拠〕

- 会員IDと1対1で対応する整理番号（例: `R000001`）を新規付与し、**顧客・購買・アクセスの3テーブルで同じ整理番号**を用いて個人単位の履歴結合を維持する。
- 元の会員IDは加工後データから除去する。
