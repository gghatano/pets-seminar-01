# 参照資料

本プロジェクトが一次情報として参照する資料を整理します。

資料そのものをリポジトリに格納する場合は、公開条件・著作権等を確認してください。
問題がある場合は URL および参照情報のみを管理します（本プロジェクトの既定は **URL・参照情報のみの管理**）。

## 著作権・格納方針

- 個人情報保護委員会（ppc.go.jp）の公表資料は、原則として政府標準利用規約に準じ、出典明示のうえで複製・再配布が可能とされる（利用時に各資料の利用条件を確認すること）。
- 本プロジェクトの既定は **URL・参照情報のみを管理**し、PDF 原本はコミットしない。原本を格納する場合は、上記利用条件を確認し、出典（資料名・公開主体・URL・参照日）を明記する。

> 対象事例（case01: 仮名加工情報）について、必ず一次情報（R1）を確認した上で設計してください。
> レポート記載事項と、教材作成のために独自補完した内容を明確に区別します。
