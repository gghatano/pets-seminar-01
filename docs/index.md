# 仮名加工情報・匿名加工情報 事例再現デモ

個人情報保護委員会事務局が公開している仮名加工情報・匿名加工情報の事例レポートを一次情報として参照し、
掲載事例のデータ加工を Python で再現する **デモ・教材** です。

!!! warning "教材としての位置づけ"
    本サイトは制度理解・加工設計プロセスの理解・Python による加工方法の理解を目的とします。
    扱うデータはすべて教材用の**合成データ（ダミーデータ）**であり、
    実際の個人データに対する法令適合性を保証するものではありません。
    レポート記載事項と、教材作成のために独自補完した内容は本文中で区別します。

## この教材の見せ方（3層構成）

本教材は「加工コードそのもの」ではなく、
**データを見て → 利用目的を考え → 情報特性を評価し → 加工方針を設計し → 実装・確認する**
という **データ加工設計の思考プロセス** を追体験してもらうことを中心に置きます。
そのため、媒体を役割で分けています。

| 層 | 媒体 | 内容 |
|----|------|------|
| **① 検討** | この HTML サイト | 事例理解 → ダミーデータ・テーブル定義 → 利用目的 → 情報特性評価 → **加工設計の決定** |
| **② 実装** | Google Colaboratory (Notebook) | 加工設計にもとづく**加工処理の実行** |
| **③ 結果** | この HTML サイト | 加工前後の比較・**確認テスト結果**・加工後データによる分析 |

## 進め方

1. **① 検討** を上から読み、なぜその加工方針になるのかを理解する
2. **② 実装（Colab）** を開き、実際に加工処理を実行する
3. **③ 結果** で、加工によって何が失われ・何が維持されたか、目的の分析が成立するかを確認する
